# Настройка Google Apps Script Webhook

## 📋 Что нужно сделать

Чтобы получать **мгновенные уведомления в Telegram** при каждой новой заявке с формы.

---

## 🔧 Пошаговая инструкция

### 1. Откройте таблицу ответов

Перейдите в Google Forms → вкладка "Ответы" → нажмите на иконку Google Sheets

**Или прямая ссылка:**
```
https://docs.google.com/spreadsheets/d/[ID_ТАБЛИЦЫ]
```

### 2. Откройте Apps Script

В таблице: **Расширения → Apps Script**

Откроется редактор кода.

### 3. Вставьте код

Скопируйте содержимое файла `apps-script-webhook.js` и вставьте в редактор.

### 4. Настройте токены

Замените в коде:

```javascript
const TELEGRAM_BOT_TOKEN = 'YOUR_BOT_TOKEN_HERE';
```

**Где взять токен:**
1. Напишите @BotFather в Telegram
2. `/newbot` → дайте имя боту
3. Скопируйте токен (выглядит как: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`)

```javascript
const TELEGRAM_CHAT_ID = '479231184'; // Already set to your ID
```

### 5. Сохраните

**Ctrl + S** или **Файл → Сохранить**

Назовите проект: "223-ФЗ Webhook"

### 6. Настройте триггер

В左侧 меню: **Триггеры** (часы) → **Добавить триггер**

Настройки:
- **Функция:** `onFormSubmit`
- **Развертывание:** `Head`
- **Источник события:** `Из таблицы`
- **Тип события:** `При отправке формы`

Нажмите **Сохранить**

### 7. Разрешите доступ

Google запросит разрешение на:
- Отправку HTTP-запросов
- Доступ к таблице

Нажмите **Разрешить**

### 8. Протестируйте

В редакторе Apps Script:
1. Выберите функцию `testWebhook` из dropdown
2. Нажмите **Выполнить**
3. Проверьте Telegram — должно прийти тестовое сообщение ✅

---

## 📊 Архитектура

```
Пользователь заполняет форму
    ↓
Google Forms сохраняет в Sheets
    ↓
Триггер onFormSubmit срабатывает
    ↓
Apps Script отправляет POST в Telegram Bot API
    ↓
Вы получаете уведомление в Telegram
```

## 🛠 troubleshooting

### ❌ "Exception: Request failed, returned code 400"

**Проблема:** Неверный токен бота или chat_id

**Решение:**
- Проверьте токен у @BotFather
- Проверьте chat_id через @userinfobot

### ❌ "Триггер не срабатывает"

**Проблема:** Триггер настроен на форму, а не на таблицу

**Решение:**
- Удалите старый триггер
- Создайте новый с источником "Из таблицы" (не "Из формы")

### ❌ "No permission"

**Проблема:** Не дали разрешение скрипту

**Решение:**
- Запустите функцию `testWebhook` вручную
- Разрешите доступ при запросе
